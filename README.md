# observing-the-weather

## Environment Variables

| Name | Purpose |
| ---- | ------- |
| WEATHER_REPORT_DIR | Overrides the default value of `/data` |
| WEATHER_OBSERVATION_STATION_ID | Overrides default value of `KSTP` |
| WEATHER_FORECAST_STATION_ID | Overrides default value of `MPX` |
| WEATHER_LOG_DIRECTORY | Overrides default value of `logs` |
| WEATHER_LOG_FILE | Overrides default value of `weatherlog.json` |
